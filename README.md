## Pitch Analyzers: 
- CLI utilities for frequency analysis
- Focuses on the 4-string violin pitch range: G3 - E7 (196.00 Hz - 2637.02 Hz)
- Built using Parselmouth and Claude Sonnet 3.5
- Coded with the assistance of Anthropic's Claude Sonnet 3.5

## Rhythm Analyzer: 
- CLI utility for rhythm analysis
- Estimates the tempo in beats per minute (BPM)
- Built with Librosa and LM Studio
- Coded with the assistance of Google's gemma-3-12b

## Timbre Analyzer
- CLI utility for timbre analysis
- Analyzes timbre of .wav audio file samples
- Built with Librosa and Claude Sonnet 4
- Coded with the assistance of Anthropic's Claude Sonnet 4
